# Claude Opus–style Chat (not affiliated)

A ready-to-paste Custom GPT configuration that emulates a Claude Opus–style conversational assistant (calm, concise, collaborative). This repository contains the system prompt, starter instructions, example conversations, recommended settings, and a helper export.json to import or reference.

Important legal/ethics note (keep this exactly):
This assistant emulates a Claude Opus–style conversational voice for experimentation and convenience. It is not affiliated with, endorsed by, or a replacement for Anthropic’s Claude.

Contents
- SYSTEM_PROMPT.md — system prompt / instructions for the Custom GPT builder
- STARTER_INSTRUCTIONS.md — example "What would you like this GPT to do?" text
- EXAMPLE_CONVERSATIONS.md — sample conversation(s)
- SETTINGS.json — suggested model settings and toggle metadata
- CHECKLIST.md — builder checklist and publishing steps
- LICENSE — MIT License
- .gitignore
- export.json — JSON package of the above pieces

Usage
1. Create a new GitHub repository (see commands below).
2. Clone locally, copy these files into the repo root, commit, and push.
3. Use SYSTEM_PROMPT.md and the other files when creating a Custom GPT in the ChatGPT builder.
